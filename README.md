# 🎵 Music Player Web Application

## 📌 Overview
A simple and interactive web-based music player that allows users to play, pause, and navigate through a playlist. Built using HTML, CSS, and JavaScript, it provides a clean and user-friendly interface for seamless audio playback.

## 🚀 Features
- Play, pause, and skip tracks
- Adjustable volume control
- Interactive progress bar
- Playlist navigation
- Real-time audio control using Web Audio API

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Web Audio API

## ⚙️ How It Works
1. Load the music playlist
2. Select a song to play
3. Use controls to play/pause/skip
4. Adjust volume and track position

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
